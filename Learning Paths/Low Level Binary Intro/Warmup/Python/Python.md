## Problem Description
Python is one of the most popular programming languages around at the time of this writing (2023). It’s a good language to get familiar with. There’s a lot of support for many tasks, available as modules to import to your own project. These Python-based challenges are meant to be an introduction to the language and how exploits work at a high level.

## Solution
This is an informational checkpoint challenge meant to highlight the importance of Python in CTFs and cybersecurity. 

Python is frequently the language of choice for writing exploit scripts, automating interactions with challenges (using libraries like `pwntools`), and quickly processing strings, encodings, and math (as you saw in the previous ASCII and Hexadecimal challenges).
