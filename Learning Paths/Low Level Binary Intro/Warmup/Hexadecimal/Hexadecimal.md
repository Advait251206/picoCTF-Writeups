## Problem Description
Hexadecimal is a number base that is common in computers. It’s slightly more compact than decimal and it divides more evenly into commonly used numbers in computing, such as 32 and 64. Hexadecimal will be used extensively in the problems in this playlist, so please use this next challenge to practice converting from hexadecimal to decimal.

## Solution
This is another checkpoint/informational challenge designed to introduce you to the hexadecimal (base-16) number system, which is fundamental in low-level computing and binary exploitation. 

Since this is purely an introductory checkpoint, there are no technical steps or scripts to run. You are simply encouraged to review how hexadecimal conversions work.

Once you are comfortable with the concept, simply click the **Mark Complete** button on the picoCTF learning path to proceed!
