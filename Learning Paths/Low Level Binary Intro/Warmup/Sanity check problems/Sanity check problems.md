## Problem Description
Pretty much every capture-the-flag competition (CTF) includes a "trivial" problem that serves as evidence that both your internet connection and the competition platform is running as expected. The next challenge is one such problem.

## Solution
This is a simple sanity check challenge intended to verify that your connection to the picoCTF platform is working correctly. 

Since this is purely an introductory checkpoint, there are no technical steps, files to download, or command-line execution (Bash or PowerShell) required for this challenge. 

All you need to do is click the **Mark Complete** button on the picoCTF learning path to proceed to the actual challenges!
