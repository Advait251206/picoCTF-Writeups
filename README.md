# 🚩 picoCTF Writeups

Welcome to my personal collection of **picoCTF Writeups**! 

This repository serves as a centralized hub for all my solutions, methodologies, and learning notes as I progress through various picoCTF challenges. Whether you're here to learn, compare solutions, or just browse, I hope you find these writeups helpful.

## 🎯 Purpose
The main goal of this repository is to document my journey in cybersecurity and capture the problem-solving processes I use for CTF challenges. Writing these helps me solidify concepts and build a personal knowledge base.

## 📂 Structure
The repository is organized by learning paths and categories to make navigation easy. Inside each challenge's folder, you will generally find:
- A `README.md` or dedicated markdown file with the full writeup.
- Any scripts (Python, Bash, etc.) used to solve the problem.
- Relevant screenshots or files provided by the challenge.

*Example:*
```text
Learning Paths/
└── General Skills in CTF's/
    └── Problem Set 1/
        └── Lets Warm Up/
            └── Lets Warm Up.md
```
