## Problem Description
For the most part, computers understand one thing: numbers. This means that everything that happens inside computers can be reduced to a number. The keys on a keyboard all have a number. The relationship between keyboard keys and their number is called the ASCII Encoding. The next challenge gives you some ASCII numbers in hexadecimal base and wants to know what the message is.

## Solution
This is another checkpoint and informational challenge. It introduces the concept of ASCII encoding, which maps numbers to characters.

Since this challenge just provides foundational knowledge for the upcoming problems (which will likely involve converting hexadecimal values to ASCII characters), there are no commands or files to execute right now.
